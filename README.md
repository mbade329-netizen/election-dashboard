# 🗳️ Election Dashboard

## 📌 Project Overview
This project is a simple Election Dashboard built using Python, Streamlit, and Matplotlib. It visualizes election results with interactive charts and analytics.

## 🚀 Features
- 📊 Election Result Dashboard
- 📈 Bar Chart Visualization
- 🏆 Leading Party Analysis
- 📋 Party-wise Vote Statistics
- 💻 Interactive Streamlit Interface

## 🛠️ Technologies Used
- Python
- Streamlit
- Matplotlib

## 📂 Project Files
- app.py (Day 1)
- day2.py (Day 2)

## ▶️ How to Run

```bash
pip install streamlit matplotlib pandas
streamlit run app.py
```

## 📸 Output
Professional Election Analytics Dashboard built for learning data visualization using python
